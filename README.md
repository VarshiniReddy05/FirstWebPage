# 🌐 Task 1 – Web Development Basics

This is a beginner-friendly HTML/CSS/JavaScript project . The goal was to build a personal webpage showcasing core front-end development skills.

---

## 📌 Features

- 🧭 **Semantic HTML Structure**  
  Cleanly structured webpage using modern HTML5 elements.

- 🎨 **CSS Styling**  
  Custom CSS styling including:
  - Responsive layout
  - Gradient headers
  - Modal popup for contact info
  - Icon integration

- 🖼️ **Media Support**  
  Integration of image elements with descriptive alt tags.

- 🔗 **Social Media Links**  
  LinkedIn and GitHub icons with links to personal profiles.

- 📨 **Interactive Contact Modal**  
  Contact button triggers a modal popup using JavaScript.

---

## 💻 Technologies Used

- **HTML5** – Structure and content
- **CSS3** – Styling and responsive design
- **JavaScript (Vanilla)** – DOM interaction and modal functionality

---

## 🧠 Learning Outcomes

- Understanding of HTML document structure
- Applying CSS for layout and styling
- Creating modals and interactive components using basic JavaScript
- Practicing clean and readable code with real-world context
